一、先搞懂：五险一金到底是哪"五险"哪"一金"？

很多人以为五险一金扣的钱都打水漂了，其实根本不是，每一笔钱都和你的生活息息相关，都是你的权益：

五险（也就是我们说的社保）

  • 养老保险：你现在交的钱，都存进你的养老账户，退休之后就能领养老金，交得越多、交得越久，退休领得越多。
  • 医疗保险：最实用的一个保险，看病、买药、住院都能报销，每个月还会有钱打进你的医保个人账户，药店买药直接刷。
  • 失业保险：被公司辞退、非本人意愿失业的，能领失业金，哪怕只交了1年社保，也能领，能帮你度过没工作的空窗期。
  • 工伤保险：工作中受伤、得了职业病，能拿到赔偿，这笔钱完全由公司交，我们自己不用出一分钱。
  • 生育保险：女生生孩子能领生育津贴、报销生育医疗费，男生也能享受陪产假和陪产津贴，现在大部分城市都合并到医保里了。

一金（住房公积金）

这绝对是五险一金里最实在的一笔钱，完全就是你的"隐形存款"：

  • 你自己扣多少，公司就要给你交多少，比如你每个月扣1000，公司也要给你交1000，这2000块都进你自己的公积金账户，完全归你所有。
  • 买房的时候能用公积金贷款，利率比商业贷款低很多，能帮你省下几十万的利息；租房、装修、大修房子也能取出来用。

所以说，五险一金不是扣你的钱，是帮你存钱、给你保障，一定要搞明白，别白白浪费了自己的权益。

二、五险一金到底扣多少？缴费比例一次性讲明白

五险一金分为个人缴费部分和单位缴费部分，我们工资里扣的，只是个人缴费的部分，单位还要给我们交大头，具体的缴费比例，全国有统一的框架，每个城市会有细微的调整，我给大家整理了最通用的标准：

险种 个人缴费比例 单位缴费比例 备注
养老保险 8% 16% 全国统一标准
医疗保险+生育保险 2% 8%-10% 各城市略有差异
失业保险 0.2%-0.5% 0.5%-1% 各城市略有差异
工伤保险 0% 0.2%-1.9% 按行业风险划分，个人不缴费
住房公积金 5%-12% 5%-12% 个人与单位同比例，可在范围内选择

这里有个重点：缴费基数。不是所有公司都按你的实际月薪当缴费基数，很多公司会按当地最低工资，或者社保缴费基数的下限给你交，这样你每个月扣的钱少了，到手工资看起来多了，但公司给你交的钱也少了，长远来看，你的养老、公积金账户都亏了。

三、手把手教你：算清五险一金扣款+到手工资

不用记复杂的公式，我用一个真实的例子，给大家一步步算明白，看完你就能自己算自己的工资了：

例子：小李在深圳工作，税前月薪10000元，公司按实际月薪作为缴费基数，公积金按10%的比例缴纳。

第一步：算个人五险一金扣款总额

  • 养老保险：10000×8% = 800元
  • 医疗保险：10000×2% = 200元
  • 失业保险：10000×0.3% = 30元
  • 公积金：10000×10% = 1000元
  • 个人五险一金总扣款：800+200+30+1000 = 2030元

第二步：算个税应纳税所得额

个税起征点是5000元/月，小李每个月还有1500元的租房专项附加扣除，所以：

应纳税所得额 = 10000 - 5000 - 2030 - 1500 = 1470元

第三步：算个税

1470元在3%的个税税率档，所以个税=1470×3% = 44.1元

第四步：算最终到手工资

到手工资 = 10000 - 2030 - 44.1 = 7925.9元

就这么简单四步，你的五险一金扣了多少、个税交了多少、到手能拿多少，就算得明明白白了。

四、90%的人都踩过的五险一金3个大坑

我身边很多朋友，包括我自己，都在五险一金上吃过亏，大家一定要避开这几个坑：

坑1：公司按最低基数交五险一金

这是最常见的坑，比如你月薪10000，公司按2360的最低工资给你交社保，你个人每个月少扣几百，公司能少交几千，看起来到手工资多了，但你的养老金账户、公积金账户每年都少了几万块，买房贷款、退休领钱的时候，亏的都是自己。

坑2：公积金按最低比例交，甚至不交

公积金的缴费比例是5%-12%，很多公司只按5%给你交，甚至有些小公司不给员工交公积金，这都是不合规的。公积金是强制缴纳的，而且你交多少公司就要补多少，按12%交和按5%交，一年下来能差出上万块，一定要和公司确认清楚。

坑3：试用期不给交五险一金

很多公司说"试用期转正之后才给交社保"，这是完全违法的！根据《社会保险法》，员工入职30天内，公司就必须给员工缴纳社保，试用期也属于劳动合同期，必须交五险一金，别被公司忽悠了。
